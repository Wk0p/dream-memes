What Core Technology do we use?

We use the following programming languages for the creation of websites: HTML, CSS, JavaScript (JS), Bootstrap and jQueary. For more information about these, please read below.

HTML

HTML, also known as Hypertext Markup Language, is a 20+ year old markup language for the creation of websites. It semantically describes the web, meaning that it allows the machine to understand what we want from it. It allows the machine to ‘interpret and compose text, images, and other material into visual or audible web pages.’

It was created by created by Sir Tim Berners-Lee in late 1991 who had the intention of it being a way to easily share information, however it officially was published in 1993, then again in 1995 with minor changes, followed by January 1997 and December 1997. With various minor updates here and there HTML5 -which is the current version was released in 2014

It is the basic of all websites, the basic layout, the structure, the start of it all. Without the HTML there can’t be any CSS or JavaScript, however it can exist on its own. CSS and JS can assist in HTML to look or act differently.

The way that HTML works is that it is based on elements, which are the building blocks of the website. Each element is made up of tags, such tags usually look something like this <p> </p>p>. Each tag has its own property - it might be how it looks, or how it is placed by default on the web. The ‘<’ opens the tag, and ‘>’ closes it, each element needs to be opened and closed with the right tag. And each opening tag, has to have a closing tag. The closing tags are a copy of the opening but have a ‘/’ in front of the first letter of the tag.

CSS

CSS or Cascading Style Sheet is a coding language that assists the creation of a website. It is what allows that one thing to be placed exactly in that one spot, it is responsible for the presentation of the HTML. It is responsible for fonts, colours, position, background, opacity and so much more.

Like HTML it is over 20 years old but CSS was developed after. It was first proposed in 1994 by Håkon Wium Lie and then published in 1996. Like the HTML code it went through various updates throughout the years

Now, since it is called a cascading style sheet, it means that the order that it is coded in is very important. As the last line of code is the newest line of code it means that it higher in hierarchy than the last. Therefore, if the code looked something like this, the font would e blue and not red

p{ Font-color: red; }
p{ Font-color: blue; }

To further breakdown that code the ‘p{‘ is the selector - it is what specifies what is going to be changed, it can also use ID and Classes which are ways to identify different elements of the HTML.

The ‘p{‘ is then followed by the declaration. ‘Font-color;’ as the name suggests, this is targeting the colour of the text and the ‘blue;’ defines what colour you want to change it to, if you want to be more specific with colour then you can use hex codes which is a 6-digit code for any colour or shade there is. Finally the ‘}’ is still part of the declaration but it is used at the very end, when you do not want to target anything else for that element.

JavaScript

Lastly JavaScript or JS is the coding language that is responsible for (Not to be confused with Java, JScript, JQueary or Javanese script.) How the website responds. Small things like the current date or time showing, that super smooth scrolling motion or the fact that when you click on a heading the site scrolls down to that section for you, that is all because of JS.
